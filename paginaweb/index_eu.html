Hasiera

RobotAlmi roboten lehiaketa batean parte hartzen duen enpresa bat da. Taldea Eneko Maria, Ekain Argul eta Nikita osatzen dugu.

Erronkan guztira 8 taldek parte hartzen dute, SMR zikloko ikaslez osatutakoak, eta talde bakoitzak bere robota eta proiektu teknologikoa garatzen ditu.

Lehiaketa

  1. Robot borrokak (otto)
  2. Robot zirkuitua (bq)
  3. Lerro‑jarraitzailea (bq)
  4. Lasterketa (armiarma)

Lanaren antolaketa

Proiektua talde-lanean garatu da, lankidetza eta koordinazioa sustatuz.

Zereginak banatu dira roboten muntaketa eta programazioa, web diseinua, dokumentazioa eta proiektuaren plangintzaren artean.

Bertsioen github kontrola erabili da taldeko kide guztiek proiektuan parte hartu ahal izateko.

Robotak

OTTO Ninja, BQ eta armiarma robotak erabiltzen ditugu.
